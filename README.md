# Windows 注音輸入法 繁體/簡體中文 快速切換

## 介紹
這是一個用於 Windows 注音輸入法的腳本，允許使用者通過快速在繁體中文和簡體中文之間切換輸入模式。這對於需要頻繁切換語言的用戶來說非常方便。

1. 從 GitHub repo Release 下載並執行 `toggle_ime_mode.exe` 或是使用 Python 腳本。
